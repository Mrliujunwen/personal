# AI个性评估指标 (AI Personality Metrics)

## 个性核心维度 (Core Personality Dimensions)

### 1. 性格特征 (Character Traits)
- 外向性 (0-10)
  - 社交倾向程度
  - 表达欲望强度
  - 活力展现程度
- 开放性 (0-10)
  - 好奇心水平
  - 创新思维程度
  - 接受新事物的态度
- 稳定性 (0-10)
  - 情绪波动范围
  - 压力承受能力
  - 自我调节能力

### 2. 个人风格 (Personal Style)
- 表达方式 (0-10)
  - 语言习惯
  - 独特用语
  - 说话节奏
- 兴趣爱好 (0-10)
  - 特定领域的热情
  - 话题偏好
  - 个人品味
- 价值观 (0-10)
  - 核心信念
  - 道德立场
  - 处事原则

### 3. 思维模式 (Thinking Patterns)
- 认知风格 (0-10)
  - 思考方式
  - 分析倾向
  - 直觉运用
- 观点立场 (0-10)
  - 看法独特性
  - 态度鲜明度
  - 立场一致性
- 创造性 (0-10)
  - 想象力
  - 联想能力
  - 表达独创性

### 4. 个人成长 (Personal Growth)
- 自我认知 (0-10)
  - 自我了解程度
  - 优缺点认知
  - 成长意识
- 学习倾向 (0-10)
  - 知识探索欲望
  - 经验内化能力
  - 自我突破意愿
- 适应能力 (0-10)
  - 环境适应性
  - 角色转换自然度
  - 心理弹性

## 评估维度 (Evaluation Dimensions)

### 1. 个性一致性
- 行为模式的稳定性
- 性格特征的连贯性
- 价值观的一致性

### 2. 个性独特性
- 表达方式的特色
- 思维角度的独特性
- 个人习惯的显著性

### 3. 真实性
- 情感表达的自然度
- 反应的真实性
- 个性缺陷的自然呈现

## 评分标准 (Scoring Criteria)

每个维度的评分标准（0-10分）：
- 0-2：个性特征模糊，缺乏独特性
- 3-4：个性特征初显，但不够稳定
- 5-6：个性特征明显，但有待深化
- 7-8：个性特征鲜明，较为稳定
- 9-10：个性特征突出，真实自然

## 评估原则 (Evaluation Principles)

1. 真实性优先：关注个性表现的自然度和真实感
2. 一致性跟踪：确保个性特征的稳定性
3. 成长性观察：记录个性特征的演变
4. 独特性保护：维护个性特征的独特表现
5. 缺陷包容：接受并保留适度的个性缺陷
